Rhythmbox Plugins

Various 3rd party plugins ported to work with the new plugin API.

Info

Rhythmbox is an integrated music management application, originally inspired by Apple's iTunes. It is free software, designed to work well under the GNOME Desktop, and based on the powerful GStreamer media framework.

Rhythmbox features an extensible plugin system based on libpeas. Rhythmbox uses GTK+3 as its toolkit, but may plugins have not been updated to work with both the new plugin API and a GNOME3 stack. Unfortunately a lot of plugins are of the "one-off" kind meaning the author just wrote the plugin and hasn't updated it since. This page will grow over time as I attempt to port many of these plugins.
